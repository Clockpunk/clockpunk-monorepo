# Руководство по редактированию контента

## Доступ к Sanity Studio

После деплоя Studio будет доступна по адресу:
- Локально: http://localhost:3333
- Production: https://ваш-проект.sanity.studio (или ваш Vercel URL)

Вход через Google или GitHub аккаунт.

## Структура контента

### 1. Site Settings (Настройки сайта)

**Singleton** - создается один раз.

Содержит глобальные настройки:
- Название сайта
- Описание
- Логотип
- OG изображение для соцсетей
- Социальные сети (Instagram, Telegram и т.д.)
- Контакты (email, телефон, адрес)
- SEO настройки по умолчанию

### 2. Home Page (Главная страница)

**Singleton** - создается один раз.

Главная страница строится из **секций** (sections). Секции управляются массивом - их можно:
- Добавлять/удалять
- Включать/выключать (переключатель **Enabled**)
- Перетаскивать для изменения порядка

### 3. Pages (Страницы)

Обычные страницы сайта:
- `/about` - об авторе
- `/contacts` - контакты
- `/legal` - юридическая информация

Каждая страница:
- Имеет уникальный slug (URL)
- Состоит из секций (как главная)
- Имеет SEO-настройки

### 4. Artworks (Работы)

Карточки работ/арт-объектов.

## Управление секциями

### Hero Section (Главный экран)

**Использование**: Первая секция на главной странице

**Поля**:
- **Enabled** - включить/выключить
- **Headline** - главный заголовок (большой текст)
- **Subheadline** - подзаголовок (меньше, под заголовком)
- **Background Image** - фоновое изображение
- **Video URL** - URL видео (альтернатива изображению, MP4)
- **Primary CTA** - основная кнопка (текст + ссылка)
- **Secondary CTA** - дополнительная кнопка

**Пример контента**:
```
Headline: ИНЖЕНЕРНАЯ ЭСТЕТИКА, СТАВШАЯ ИСКУССТВОМ
Subheadline: Где механизм становится смыслом
Primary CTA: Смотреть работы → /artworks
```

### Featured Artworks Section (Избранные работы)

**Использование**: Показ избранных работ на главной

**Режимы**:
1. **Автоматический** (autoFeatured = true):
   - Показывает работы с флагом "Избранная работа"
   - Сортировка: sortOrder → год → название
   - Количество: указывается в **Show Count**

2. **Ручной** (autoFeatured = false):
   - Выбираете конкретные работы через **Items**
   - Показывает в том порядке, в котором добавлены

**Поля**:
- **Title** - заголовок секции
- **Subtitle** - подзаголовок (опционально)
- **Auto Featured** - режим выбора работ
- **Items** - ручной выбор работ (если Auto = false)
- **Show Count** - количество работ (1-12)

**Пример контента**:
```
Title: Избранные работы
Subtitle: Коллекция механической эстетики
Auto Featured: true
Show Count: 6
```

### Gallery Grid Section (Галерея)

**Использование**: Показ конкретного набора работ

**Отличие от Featured**: Всегда ручной выбор работ

**Поля**:
- **Title** - заголовок секции
- **Subtitle** - подзаголовок
- **Artworks** - выбор работ через reference

**Пример контента**:
```
Title: Automotive Collection
Subtitle: Автомобильная серия
Artworks: [DeLorean, BMW X6 M, Toyota Land Cruiser]
```

### Brand Story Section (История бренда)

**Использование**: Текстовый блок с изображением

**Поля**:
- **Title** - заголовок
- **Body** - rich text (форматированный текст)
- **Image** - изображение (опционально)

**Макет**:
- Если есть изображение: 2 колонки (текст + изображение)
- Если нет изображения: 1 колонка по центру

**Пример контента**:
```
Title: Философия Clockpunk
Body: Clockpunk — это система, где деталь важнее формы. 
Каждая шестеренка — не декор, а часть единой истории...
```

### Process Section (Процесс создания)

**Использование**: Пошаговое описание процесса

**Поля**:
- **Title** - заголовок секции
- **Subtitle** - подзаголовок
- **Items** - массив этапов:
  - **Title** - название этапа
  - **Text** - описание этапа
  - **Icon** - иконка или emoji (опционально)

**Пример контента**:
```
Title: Технология создания
Items:
  1. Реальные механизмы
     Использование настоящих деталей техники и часов
  2. Ручная сборка
     Сложная архитектура слоев создает объем
  3. Инженерная точность
     Математический расчет композиции
  4. Ювелирная обработка
     Финальная шлифовка каждого элемента
```

### Press Section (Пресса и выставки)

**Использование**: Публикации, выставки, награды

**Поля**:
- **Title** - заголовок
- **Subtitle** - подзаголовок
- **Items** - массив событий:
  - **Title** - название публикации/выставки
  - **Place** - где (издание/место)
  - **Date** - дата
  - **Link** - ссылка (опционально)

**Пример контента**:
```
Title: Выставки и публикации
Items:
  - Выставка современного искусства
    ЦВЗ "Манеж"
    15-30 декабря 2024
```

### CTA Section (Призыв к действию)

**Использование**: Финальный блок с призывом

**Поля**:
- **Title** - заголовок
- **Text** - дополнительный текст
- **CTA** - кнопка:
  - **Label** - текст кнопки
  - **Href** - ссылка

**Пример контента**:
```
Title: Создайте свою историю
Text: Эксклюзивные персонализированные творения
CTA: Связаться с автором → /contacts
```

## Управление работами (Artworks)

### Создание работы

1. Перейдите в **Artworks** → **Create new**
2. Заполните обязательные поля:
   - Название
   - Slug (автоматически из названия)
   - Статус
   - Год
   - Обложка (coverImage)

### Статусы работ

- **Draft** (Черновик) - не показывается на сайте
- **Available** (Доступно) - показывается в каталоге
- **Sold** (Продано) - показывается с меткой "Продано"
- **Archived** (Архив) - не показывается на сайте

### Сортировка и приоритет

**Порядок показа работ** (от высшего к низшему приоритету):
1. **Featured** (Избранная работа) - да/нет
2. **Sort Order** - чем меньше число, тем выше
3. **Year** - новые первыми
4. **Title** - по алфавиту

**Примеры**:
```
Работа A: featured=true, sortOrder=1, year=2024  → показ 1-й
Работа B: featured=true, sortOrder=2, year=2024  → показ 2-й
Работа C: featured=false, sortOrder=null, year=2024  → показ 3-й
```

### Изображения

**Требования**:
- **Обложка (Cover Image)**: обязательно, главное изображение работы
- **Галерея (Images)**: опционально, дополнительные виды

**Рекомендации**:
- Минимальный размер: 1200×900 px
- Формат: JPG или PNG
- Соотношение сторон: 4:3 (предпочтительно)
- **ОБЯЗАТЕЛЬНО**: заполняйте Alt Text для каждого изображения

**Alt Text**:
```
Хорошо: "Clockpunk интерпретация DeLorean - вид спереди"
Плохо: "IMG_1234" или пустое поле
```

### Описание работы

Используйте Rich Text редактор:
- **Абзацы** - обычный текст
- **H3, H4** - подзаголовки
- **Strong** - жирный текст
- **Em** - курсив

**Пример структуры**:
```
### О работе
Описание концепции и идеи работы.

### Технические детали
Количество использованных деталей, особенности сборки.

### Символизм
Что означает каждый элемент.
```

### Цена

**Варианты**:
1. **Указать цену**: заполните **Price** + выберите **Currency**
2. **Цена по запросу**: включите **Price On Request**

Если включен "Цена по запросу", поле Price игнорируется.

### Теги

Используйте теги для категоризации:
```
Примеры: automotive, weapon, industrial, limited-edition, sold-out
```

## SEO настройки

### Для страниц (Pages)

В разделе **SEO** каждой страницы:
- **Meta Title** - заголовок в поисковике (до 60 символов)
- **Meta Description** - описание в поисковике (до 160 символов)
- **OG Image** - изображение для соцсетей (1200×630 px)
- **Noindex** - скрыть от индексации (для тестовых страниц)

**Если не заполнено**: используются дефолтные значения из Site Settings.

### Для работ (Artworks)

Аналогично страницам, опциональный блок SEO.

**Автоматические значения** (если не заполнено):
- Meta Title: `{Название работы} - Clockpunk`
- Meta Description: `{Год} • {Материалы}`
- OG Image: обложка работы

## Частые сценарии

### Изменить порядок секций на главной

1. Откройте **Home Page**
2. В блоке **Sections** перетащите секции мышкой
3. Сохраните (Publish)

### Временно скрыть секцию

1. Откройте нужную страницу
2. Найдите секцию в списке
3. Выключите переключатель **Enabled**
4. Сохраните

Секция останется в списке, но не будет показываться на сайте.

### Добавить работу в избранные

1. Откройте работу в **Artworks**
2. Включите **Featured**
3. Опционально: установите **Sort Order** (чем меньше, тем выше)
4. Сохраните

Работа автоматически появится в секциях с **Auto Featured = true**.

### Создать новую страницу

1. Перейдите в **Pages** → **Create new**
2. Заполните:
   - **Title**: название страницы
   - **Slug**: URL (например, `services` → /services)
3. Добавьте секции
4. Настройте SEO
5. Сохраните и опубликуйте

Страница будет доступна по адресу: `/{slug}`

### Обновить контакты

1. Откройте **Site Settings**
2. В блоке **Contacts** обновите поля
3. Сохраните

Контакты используются:
- На странице /contacts (если нет custom page)
- В футере (если добавлен)

## Предпросмотр изменений

Sanity Studio:
- Изменения сохраняются сразу при Publish
- Кеш сайта обновляется в течение 3 минут (revalidate = 180 сек)

Для немедленного обновления:
- Перезагрузите страницу сайта через 3 минуты
- Или очистите кеш браузера (Ctrl+Shift+R)

## Backup и версионирование

Sanity автоматически сохраняет историю изменений:
- Каждая публикация = новая версия
- Откатиться к предыдущей версии: **History** → выберите версию → **Restore**

## Права доступа

Настройка прав в Sanity Dashboard:
1. Перейдите в [sanity.io/manage](https://sanity.io/manage)
2. Выберите проект
3. **Members** → добавьте пользователей
4. Роли:
   - **Administrator** - полный доступ
   - **Editor** - может редактировать контент
   - **Viewer** - только просмотр

## Технические ограничения

- **Изображения**: до 100 МБ
- **Секций на странице**: до 20 (рекомендуется)
- **Работ в секции**: до 12
- **Этапов процесса**: до 8

## Помощь

При возникновении вопросов:
1. Проверьте, что все обязательные поля заполнены
2. Убедитесь, что изменения опубликованы (Publish)
3. Дождитесь обновления кеша (3 минуты)
4. Проверьте статус работ (Draft не показывается)
